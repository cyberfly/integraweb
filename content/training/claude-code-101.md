Every series has a beginning. **Claude Code 101 — First Edition** was ours — and it sold out.

On April 2, 2026, **157 programmers** from across Malaysia joined the inaugural cohort to turn Claude Code into a genuine coding superpower. Both ticket tiers — early bird and general — sold out ahead of the live Zoom session.

Across the evening, participants worked through the full curriculum that would define every later edition: configuring **Memory** so Claude understands your project, using **Plan Mode** to research before executing, integrating external tools through **MCP**, automating repetitive work with **Slash Commands**, **Skills**, **Hooks**, and **Plugins**, and running parallel **Subagents** across Git **Worktrees**. The session closed with live demonstrations of real workflows — `/simplify` for code-quality review, `/security-review` for vulnerability scanning, `/rewind` for safe undo, and `/voice` for hands-free commands.

The response was strong enough to turn a one-off workshop into a recurring program, refined edition after edition into the advanced curriculum we run today. For the most complete, up-to-date version of this training, see the [Third Edition](/training/claude-code-101-third-edition.html).
